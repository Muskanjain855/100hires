## Tools Installed

Cursor IDE — AI-powered code editor, downloaded from cursor.com
Claude Code extension — Anthropic's AI coding assistant, installed via Cursor's Extensions panel
Codex extension — OpenAI's code completion tool, installed via Cursor's Extensions panel
Git — Version control system, installed to enable cloning and pushing to GitHub

## Steps Completed

Downloaded and installed Cursor IDE from cursor.com
Opened the Extensions panel in Cursor (Ctrl+Shift+X / Cmd+Shift+X) and searched for "Claude Code" — installed and logged in
Searched for "Codex" in the same Extensions panel — installed and logged in
Installed Git to enable version control and GitHub integration
Created a new public repository on github.com
Cloned the repository locally using Cursor's built-in Clone Repository option
Created this README.md to document the process
Committed and pushed changes to GitHub

## Issues Encountered & How I Solved Them

Difficulty finding the Claude Code extension — When searching "Claude Code" in the Extensions panel, multiple results appeared and it was not immediately obvious which one to install. Looked more carefully at the publisher name (Anthropic) to identify the correct extension.
Two modes in Claude Code — Agent mode vs Anthropic mode — After installing the Claude Code extension, there were two available modes: Agent mode and Anthropic mode. Agent mode allows Claude to take multi-step actions autonomously (like editing files, running commands), while Anthropic mode is a more standard chat-based assistant. Used Agent mode for hands-on coding assistance.
